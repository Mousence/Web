Типы данных в языке C++

Переменная - это именованная область памяти, содержимое которой может изменяться в процессе выполнения программы. Прежде чем использовать переменную, ее нужно объявить. Для объявления переменной нужно указать ее тип и имя следующим образом:

		
			type name;

type (тип переменной), определяет три вещи:

  1. Сколько памяти переменная будет занимать;
  2. Какие значения она сможет принимать;
  3. Какие операции над ней можно выполнять;

Все типы данных в языке C++ можно разделить на следующие категории:

  1. Логические типы. Включает в себя всего один тип данных – bool (Boolean). Переменная типа bool занимает 1 Байт памяти, и принимает одно из двух значений: true или false. true хранится в памяти как 1, а false как 0. На самом деле 0 – это false, а true – это все что НЕ 0;
